# Quick Start Guide for MOHID-Lagrangian

These instructions outline the minimum requirements to run the MOHID-Lagrangian program. This release includes the executable file, the pre-processing and post-processing scripts, as well as a sample RunCase.

## Installation and Execution Steps:

1. Download the `MOHID-Lagrangian_windows.zip` file from github.com and extract it into a local directory.
2. Install Miniconda from the following link: [https://www.anaconda.com/download](https://www.anaconda.com/download)
3. Open an **Anaconda Prompt** terminal.
4. Navigate to the `MOHID-Lagrangian\buildScripts` directory and execute the `MakeEnvMOHIDLagrangian.bat` file.
5. In the same Anaconda Prompt terminal, navigate to the `MOHID-Lagrangian\RUN_Cases\Arousa_2D_test_case` directory and execute the `RunCase.bat` file.

Upon completing these steps, MOHID-Lagrangian will run, and the results will be generated in the following path:
`MOHID-Lagrangian\RUN_Cases\Arousa_2D_test_case\Arousa2D_case_out`
