# Quickstart Guides: Executable and Source Installation

## 🇺🇸 English

The following directories contain comprehensive **Quickstart Guides** designed to provide a fully operational environment for both **Windows** and **Linux** platforms. Please note the fundamental differences in the deployment process for each operating system:

### **Windows Installation**
On Windows systems, the provided package includes a **pre-compiled executable**. This streamlined setup incorporates:
* The primary operational executable.
* Dedicated scripts for both **pre-processing** and **post-processing** tasks.
* A complete demonstration example to verify system functionality.

### **Linux Installation**
For Linux environments, the deployment follows a source-based approach. Users will download the **complete project repository**, which necessitates **compiling the source code** locally to generate the operational binary. This process is managed via the automated installation scripts provided in the directory, which handle dependency resolution and environment configuration.

## 🇵🇹 Português

Os seguintes diretórios contêm **Guias de Início Rápido** abrangentes, projetados para fornecer um ambiente operacional completo para as plataformas **Windows** e **Linux**. Observe as diferenças fundamentais no processo de implantação para cada sistema operacional:

### **Instalação em Windows**
Nos sistemas Windows, o pacote fornecido inclui um **executável pré-compilado**. Esta configuração otimizada incorpora:
* O executável operacional principal.
* Scripts dedicados para tarefas de **pré-processamento** e **pós-processamento**.

### **Instalação em Linux**
Para ambientes Linux, a implementação segue uma abordagem baseada em código-fonte. Os usuários descarregarão o **repositório completo do projeto**, o que exige a **compilação do código-fonte** localmente para gerar o binário operacional. Este processo é gerido através dos scripts de instalação automatizados fornecidos no diretório, que tratam da resolução de dependências e da configuração do ambiente.

Los siguientes directorios contienen **Guías de Inicio Rápido** exhaustivas diseñadas para proporcionar un entorno operativo completo tanto para plataformas **Windows** como **Linux**. Tenga en cuenta las diferencias fundamentales en el proceso de despliegue para cada sistema operativo:

### **Instalación en Windows**
En sistemas Windows, el paquete proporcionado incluye un **ejecutable pre-compilado**. Esta configuración optimizada incorpora:

Para entornos Linux, el despliegue sigue un enfoque basado en código fuente. Los usuarios descargarán el **proyecto completo**, lo que requiere la **compilación del código** localmente para generar el binario operativo. Este proceso se gestiona a través de los scripts de instalación automatizados incluidos en el directorio, los cuales se encargan de la resolución de dependencias y la configuración del entorno.
